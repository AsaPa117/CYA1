# CYA1
Career Development Experience

Description:
Studying for a job certification is my chosen adventure, as I'm aiming to become a Network Engineer. To achieve this goal, I applied and committed to a program that will prepare me to take the CompTIA Network+ certification exam sometime this summer. The program provides lecture notes, videos, questions, quizzes, and practice exams that help me build foundational knowledge in network infrastructure. 

Context: 
My goal is to become a Network Engineer with an understanding of IT infrastructure. Since this school year, I've been searching for internships that help me advance my career, but they require or highly value skills that my previous college courses don't offer. After applying for an internship and some interviews, I realized that much of what I learned at WSU isn't what my employers are looking for, so I decided to apply to a program to develop the specific skill sets that every position I apply for highly values.

Summary: 
My deliverables include my enrollment, progress, and participation in the CompTIA Network+ Certification. The course itself includes content, practice exams, and quizzes covering networking, its implementation, network operations, security, and, most importantly, troubleshooting. Although I still need to look for a suitable lab to gain hands-on experience, the program includes content that helps build foundational knowledge.
